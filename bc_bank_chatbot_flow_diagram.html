🚀 Core Components

  • chatbot_app.py: Main application entry point
  • session_manager.py: Manages chat sessions and ChromaDB connection
  • ui_utils.py: Streamlit UI configuration and styling
  • query_processor.py: Handles query analysis and routing
  • gemini_api.py: AI response generation

🧠 Intelligence Features

  • Semantic Search: Vector-based document retrieval
  • Context Awareness: Maintains conversation history
  • Follow-up Detection: Recognizes related questions
  • Category Mapping: Intelligent query classification
  • Quality Filtering: Ensures relevant responses

💾 Data Management

  • ChromaDB: Cloud-based vector database
  • CSV Processing: Banking FAQ data ingestion
  • Embeddings: Sentence transformer models
  • Caching: Query result optimization
  • Metadata: Category-based filtering

🎯 Banking Categories

  • Accounts: Savings, current, statements
  • Cards: Credit, debit, ATM services
  • Transfers: NEFT, RTGS, IMPS, UPI
  • Loans: Personal, home, car loans
  • Insurance: Life, health policies
  • Investments: FD, RD, mutual funds
  • Security: Login, fraud protection
